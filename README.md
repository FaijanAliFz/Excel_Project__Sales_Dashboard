# 🚗 Vehicle Sales Dashboard

## 📊 Overview
This project presents an interactive dashboard to analyze vehicle sales performance across different dimensions such as time, product (car make & model), and sales personnel. It helps in understanding trends, identifying top performers, and supporting data-driven decisions.

## 🎯 Objectives
- Track overall sales performance and key metrics  
- Analyze sales trends over time (monthly & yearly)  
- Identify top-performing car models and salespeople  
- Evaluate productivity across different sales segments  

## ⚙️ Features
- Interactive dashboard with filters and drill-down capabilities  
- KPI tracking for sales performance  
- Product-level analysis (car make & model)  
- Salesperson performance insights  
- Time-based trend analysis  

## 🛠️ Tools & Technologies
- Excel  
- Data Visualization  
- Pivot Tables 

## 📈 Key Insights
- Top-performing vehicles contributing maximum revenue  
- High-performing sales representatives identified  
- Seasonal trends observed in sales data  
- Better visibility into overall business performance  

## 🚀 Outcome
Developed a centralized and interactive dashboard that enables efficient monitoring of sales data and supports informed business decision-making.
